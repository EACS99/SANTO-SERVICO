# 📖 Sobre o Projeto

O **Santo Serviço** é uma aplicação desenvolvida para automatizar e organizar escalas de serviço litúrgico de forma prática, moderna, responsiva e inteligente.

O sistema aplica automaticamente regras litúrgicas reais da Igreja Católica, permitindo gerar:

* Escalas para missas específicas
* Escalas mensais automáticas
* Distribuição inteligente de funções
* Temas litúrgicos automáticos
* Controle inteligente de exclusividade
* Regras especiais para celebrações festivas
* Regras específicas para missas com bispo

---

# 🚀 Funcionalidades

## ✅ Modos de Escala

### 🎯 Missa Específica

Permite gerar escala individual para uma celebração específica.

### 📅 Escala Mensal

Geração automática de todas as celebrações do mês com regras litúrgicas dinâmicas.

# 📌 Regras de Exclusividade

## Coroinhas

* Quem faz **Naveta** não repete função
* Quem faz **Mitra** não repete função
* Quem faz **Báculo** não repete função

## Cerimoniários

* Quem faz **Turíbulo** não repete função
* Quem faz **Missal** não repete função
