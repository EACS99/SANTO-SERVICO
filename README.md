# ✨ Santo Serviço

Sistema inteligente de geração de escalas litúrgicas para coroinhas, cerimoniários e pastorais.

---

## 📖 Sobre o Projeto

O **Santo Serviço** é uma aplicação desenvolvida para automatizar e organizar escalas de serviço litúrgico de forma prática, moderna e responsiva.

O sistema permite gerar:

- Escalas para missas específicas
- Escalas mensais automáticas
- Distribuição inteligente de funções
- Temas litúrgicos automáticos
- Controle de funções exclusivas
- Regras especiais para celebrações festivas

---

# 🚀 Funcionalidades

## ✅ Modos de Escala

### 🎯 Missa Específica
Permite gerar escala individual para uma celebração.

### 📅 Escala Mensal
Geração automática de todas as celebrações do mês.

# 🧠 Inteligência do Sorteio

O sistema evita repetições desnecessárias e respeita regras especiais:

## Coroinhas

- Quem faz **Naveta** não repete função
- Quem faz **Mitra** não repete função
- Quem faz **Báculo** não repete função

## Cerimoniários

- Quem faz **Turíbulo** não repete função
- Quem faz **Missal** não repete função
- Em missas com bispo:
  - Missal Bispo não repete
  - Móvel Bispo não repete
